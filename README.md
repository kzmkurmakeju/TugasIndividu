# TugasIndividu
TUGAS INDIVIDU PBO

Sistem ini dirancang menggunakan konsep Object Oriented Programming (OOP) dengan menerapkan encapsulation. 
Setiap class memiliki atribut yang bersifat private (-) untuk menjaga keamanan data, serta method getter dan setter (+) untuk mengakses dan mengubah nilai atribut.
Class digunakan untuk merepresentasikan objek dalam sistem, misalnya data utama yang dikelola oleh program. Atribut dalam class menyimpan informasi penting, sedangkan method digunakan untuk melakukan operasi seperti mengambil data, mengubah data, atau menampilkan informasi.
